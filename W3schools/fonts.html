CSS Fonts

Fonte com serifa — Times New Roman é um exemplo clássico de fonte com serifas.

Fonte sem serifa — Arial é uma fonte limpa e moderna, ideal para telas.

Fonte monoespaçada — Courier New é muito usada para código e tabulações.

Texto grande — Usado para destacar ou títulos secundários.

Texto pequeno — Pode ser usado para notas ou legendas.
