Centralização com CSS

1. Alinhamento Horizontal com:
margin: 0 auto;

Este bloco está centralizado horizontalmente.

Usamos margin: 0 auto; para centralizar um bloco com largura definida.

2. Alinhamento com Flexbox

Este bloco está centralizado horizontal e verticalmente com Flexbox.

Com:
display: flex;
justify-content: center;
align-items: center;
é possível alinhar o conteúdo dentro do bloco.

3. Alinhamento com Grid

Centralizado com CSS Grid.

CSS Grid permite centralizar com poucas linhas usando:
place-items: center;
